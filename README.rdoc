== README

This application is used for search people on the twitter and find keywords on the bing and add in the database.

Through twitter search we can find inserted user name and his screename and add in to database through add link and we can also create manually people and add into database.

and same thing for bing , here we can find keyword through bing api and add into database we can also add manually keyword in database… in case of keyword if keyword already exist in database than it will not search on web but it will display data…

here we can also do CRUD operation on the database…
